ABOUT

SUMMARIZER

Briefer provides a summary of input to the user in the form or either text or via speech. It can take input as text, speech or document. Text summarization will be achieved.

OPTIMIZER

Briefer renders scope for optimization of text such as most frequent words, spelling corrections, grammar corrections, etc.

SENTIMENT ANALYZER

Briefer provides sentiment analysis which determines the inclination of the text either positive, negative or neutral.

LANGUAGE TRANSLATOR

As Briefer will cater to a large audience from around the globe hence it also allows users to translate the summary into their desired language.
